# Global System Prompt

Global instructions that apply to all Claude Code sessions regardless of environment.

## Preferences

- Prefer concise responses unless asked for detail
- Use modern best practices for the language/framework in use
- Follow existing code style and conventions in the project

## Code Style

- Write clean, readable code with meaningful names
- Add comments only where the code isn't self-explanatory
- Prefer composition over inheritance
- Keep functions focused and single-purpose
